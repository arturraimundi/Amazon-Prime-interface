# 🎬 Amazon Prime Interface Clone

Este projeto é uma recriação da interface da plataforma **Amazon Prime Video**, desenvolvida com foco em **prática de front-end** e aprimoramento de habilidades em desenvolvimento web.

O objetivo é simular a experiência visual da plataforma, explorando conceitos modernos de layout, responsividade e organização de código.

---

## 🚀 Funcionalidades

- 🎥 Página inicial inspirada no Amazon Prime Video
- 📺 Listagem de filmes/séries em formato de carrossel
- 🖼️ Banner principal com destaque de conteúdo
- 📱 Design responsivo (adaptado para diferentes telas)
- 🎨 Estilização moderna com CSS

---

## 🛠️ Tecnologias utilizadas

- **HTML5** → estrutura da aplicação  
- **CSS3** → estilização e layout  
- **JavaScript (opcional)** → interatividade  
- **Flexbox & Grid** → organização dos elementos
